    scrabble oracle

    a scrabble solver

    how to use

    1. click a tile then input letters to recreate your IRL scrabble board
    2. in the input field, type your rack letters
    3. in the dropdown, specify the number of blanks in your rack
    see the highest scoring options!

    advanced
    • while a tile is selected, type 1-9 or the return key to toggle the typing direction between horizontal and vertical.
    • while a tile is selected, type 0 to make the selected letter lowercase. this tells scrabble oracle that the letter was played with a blank tile, and is therefore worth 0 points.
